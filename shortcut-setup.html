Set up the Background Sounds Shortcut

DeadlineGhost controls macOS Background Sounds through a small Apple Shortcut named DeadlineGhost Background Sounds. Routing through Shortcuts is what makes the app App Sandbox–safe and App Store distributable — sandboxed apps can't write to other apps' system preferences directly. You only set this up once.

A. One-tap install (recommended)

Click the button below. macOS opens Shortcuts.app and asks you to add the shortcut to your library. Keep the name as DeadlineGhost Background Sounds — DeadlineGhost looks it up by name.

Install Shortcut from iCloud Build it yourself instead
Already installed? Open DeadlineGhost, then in Settings → Sound, the Shortcut row shows Installed. If the in-app prompt keeps appearing, click Already installed there to mark it.

B. Build it yourself

Open Shortcuts.app, create a new shortcut, name it exactly DeadlineGhost Background Sounds, and assemble these blocks in order. The screenshot below shows the finished layout.

  1. Receive Apps and Text from Nowhere. If there's no input: Continue.
  2. If Shortcut Input contains vol:
    • Replace vol: with empty in Shortcut Input → Updated Text
    • Get Numbers from Updated Text → Numbers
    • Turn background sounds On
    • Change background sounds volume to Numbers
  3. Otherwise
    • If Shortcut Input is on
      • Turn background sounds On
    • Otherwise
      • Turn background sounds Off
    • End If
  4. End If
  5. Stop and output If Result. If there's nowhere to output: Do Nothing.

C. Inputs the app sends

Every focus-session transition fires the shortcut once with one of these text inputs:

InputEffect
onTurn background sounds ON (volume unchanged)
offTurn background sounds OFF
vol:50Turn ON at 50% volume (any number 0–100)

D. Troubleshooting

  • Shortcut name mismatch. The app invokes by name. If you renamed it, restore DeadlineGhost Background Sounds exactly — capitalisation included.
  • Picking which sound plays. macOS Shortcuts' Change Background Sound action only accepts a fixed picker value, not a magic variable. Choose the sound (rain, stream, ocean, etc.) once in System Settings → Accessibility → Audio → Background Sounds. The app's Settings → Sound → Choose Sound… button opens that pane.
  • Shortcut prompt keeps reappearing. Open DeadlineGhost → Settings → Sound. The Shortcut row has a Reset button to clear the install flag. Click Already installed next time the prompt fires.
  • Volume changes get dropped. The app collapses on/off and volume into a single vol:N invocation to avoid back-to-back shortcut races. If you customised the shortcut, keep the single-pass branching above.
